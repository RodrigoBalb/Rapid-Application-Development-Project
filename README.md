Protótipo de um sistema avaliador universitario, cujo objetivo é pegar as notas dos alunos de diferentes cursos e subir pro sistema academico
Nosso aplicativo propõe-se a auxiliar o sistema interno universitario a automatizar as correções das avaliações dos cursos e armazenar no banco de dados diferentes questões academicas pra facilitar a aplicação da prova, sendo filtradas por curso e diferentes modalidades(Presencial, semipresencial e online)

Atualizado o Banco de dados, adicionando duas novas tabelas sendo elas : ADS (para o curso de Analise e desenvolvimento de sistemas) e CieComp ( para o curso de Ciencia de computação)
